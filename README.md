# MDTECHBlog Project
This project has as main idea the creation of a technology blog page. Only CSS and HTML languages ​​were used. The page has a home page, and sections for games, computers and cell phones. It is possible to navigate between the pages through the hyperlinks present both in the header and in the side area. Last but not least is the potential subscriber registration section.

# Proyecto MDTECHBlog
Este proyecto tiene como idea principal la creación de una página de blog de tecnología. Solo se utilizaron lenguajes CSS y HTML. La página tiene una página de inicio y secciones para juegos, computadoras y teléfonos celulares. Es posible navegar entre las páginas a través de los hipervínculos presentes tanto en el encabezado como en el área lateral. Por último, pero no menos importante, está la sección de registro de suscriptores potenciales.

# Projeto MDTECHBlog

A página conta com uma home, e com seções de jogos, informática e celulares. É possível navegar entre as páginas através dos hiperlinks presentes tanto no cabeçalho como na área lateral.
